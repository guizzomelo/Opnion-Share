
# Opnion-Share

<h4>Este projeto foi criado com a intenção de evoluir minhas habilidades em JavaScript e ReactJS para outro patamar. Eu apilidei este projeto de Opinion Share. O objetivo é ser uma espécie de rede social onde as pessoas podem discutir e dar suas opiniões sobre vídeos do Youtube em um chat global. Nesta aplicação é possível comentar nas postagens feitas por outros usuários, que entram na aplicação após fazer login ou criar uma conta. O projeto ainda não está 100% concluido já que eu só utilizei ferramentas front-end com html, css, javascript/React, com está aplicação eu levei o javascript/front-end ao limite, o proximo passo é implementar o back-end da aplicação usando o NodeJS.</h4>

<h3>Pagina Principal</h3>
<img width="1274" height="579" alt="Captura de tela 2025-11-28 104442" src="https://github.com/user-attachments/assets/7ca9cfad-0656-4bad-9ee8-a81bc106af2c" />

<h4>Após o login o usuário tem acesso à página principal que consiste em um chat global onde usuários podem dar suas opiniões e comentar, no lado esquerdo temos uma sidebar onde os usuários podem modificar o perfil, ver seus posts salvos como favoritos ou posts onde eles adicionaram o like, seus comentários salvos e configurações do app. A maioria das opções ainda não estão prontas pois preciso integrar a aplicação ao NodeJS para concluir, como eu falei levei o javascript a um outro nivel nesta aplicação.</h4>

<h3>Barra de pesquisa</h3>
<img width="1268" height="583" alt="Captura de tela 2025-11-28 104530" src="https://github.com/user-attachments/assets/7bc26a81-ce4a-4ca2-a02e-cf3ee6fb0ce9" />

<h4>Pesquise sobre um vídeo do Youtube na "Search bar" e a API do Youtube entrará em ação pesquisando os 5 primeiros vídeos mais relevantes referentes a sua busca, ao clicar em um vídeo um modal se abre permitindo você comentar algo sobre o vídeo ao clicar em "Share Opinion" seu vídeo será redirecionado ao chat global para todos verem. </h4>
